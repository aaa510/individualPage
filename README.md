# 个人主页

这是一个从零开始搭建的像素风个人主页，可以直接用浏览器打开 `index.html` 预览。

## 修改位置

- 在 `index.html` 中修改安杨可欣的个人情况、校园经历、实习经历和荣誉奖项。
- 在 `styles.css` 中调整像素风视觉、全屏分节、滚动切换和响应式样式。

## 本地预览

直接双击 `index.html`，或在浏览器地址栏打开该文件即可。

如果浏览器直接打开失败，可以在当前目录运行：

```powershell
python -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/index.html
```
